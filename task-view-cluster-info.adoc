[role="lead"]
您可以直接从NetApp Console查看容量信息以及许可信息。



== 查看容量信息

当您从“系统”页面选择本地ONTAP集群时，“系统详细信息”页面将显示集群的容量信息，包括总容量、已用容量和可用容量。



== 查看许可证和合同详情

您可以从“订阅和许可证”页面查看每个本地ONTAP集群的合同详情。
